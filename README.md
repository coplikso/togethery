# togethery

> Watch together. Chat together. Stay focus together.

**togetherly** is a website that can be used for studying together with friends, you could be the host and your friend just joined your room with the spesial code! uses real time connection so you and your friend don't miss a thing for study!

the web is on **live demo** now : [togethery.vercel.app](https://togethery.vercel.app)

## features
- **room based session**, create a room and share the code with your friend so they/he/she/it can study or watch together with you!
- **synced youtube playback**, watch youtube video in real time with your friends! whether its a educational video or a k-pop boy band that have 5 member!!
- **synced PDF viewer**, Review PDF file that your teachers gave you, open it in togetherly and view it on real time!!
- **Live chat**, chit chator maybe even discuss about the video or the pdf with your partners, (pls don't do gosip, its not cool)
 ---



 ## tech stack

 **Frontend** = JavaScript, CSS, HTML
 **Backend**  = Node.js (WebSocket server)
 **Realtime** = WebSocket
 **Hosting** = vercel

 ## how its works
1. **Create a room** — A unique room ID is generated for your session
2. **Share the link** — Send the room URL to friends
3. **Pick content** — Paste a YouTube URL or upload a PDF
4. **Watch/read together** — All playback controls and page navigation are broadcast to everyone in the room via WebSockets in real time
5. **Chat** — Send messages visible to all room members instantly
